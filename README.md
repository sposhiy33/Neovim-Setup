This is my setup for neovim, mostly inspired by [Typecraft's](https://www.youtube.com/watch?v=zHTeCSVAFNY&list=PLsz00TDipIffreIaUNk64KxTIkQaGguqn&pp=iAQB) setup. This repository is mainly meant to quickly migrate my neovim setup to any new Ubuntu systems I may be using.

Note: this setup was done on an AMD system running Ubuntu 22.04 LTS
Usage:
  - Download neovim
  - clone this repo within the ``~/.config/nvim`` folder
  - open neovim to begin installation of all plugins
